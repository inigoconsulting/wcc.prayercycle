wcc.prayercycle Installation
----------------------------

To install wcc.prayercycle using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wcc.prayercycle`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wcc.prayercycle
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
